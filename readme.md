## Requirements

- [Node.js](https://nodejs.org)

## Installation

Install dependencies via:

```sh
npm install
```

## Usage

Server with live reloading

```sh
npm run develop
```

Testing / Linting

```sh
npm test
```

Build files for production

```sh
npm run build
```
